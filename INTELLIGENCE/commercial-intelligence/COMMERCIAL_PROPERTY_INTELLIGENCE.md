## SECTION 8 — LEAD SCORING FRAMEWORK

Score each commercial property opportunity on a 1–10 scale:

| Factor | Weight | Scoring Criteria |
|---|---|---|
| Portfolio Size | 25% | 10+ properties = 10; 1 property = 1 |
| Decision Maker Access | 20% | Direct owner = 10; corporate procurement = 3 |
| Urgency | 20% | Code violation/fire = 10; routine = 3 |
| Contract Value Potential | 20% | $100K+/yr = 10; <$10K = 1 |
| Competition Level | 15% | No incumbent = 10; entrenched incumbent = 2 |

### Priority Targets (Score 7+)

1. Property managers with 10+ properties and no current vendor
2. Fire-damaged properties needing immediate cleanup
3. Properties with active code violations
4. New ownership (within 90 days) — no incumbent vendor
5. REITs with local property management offices
