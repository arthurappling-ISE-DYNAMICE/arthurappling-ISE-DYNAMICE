## PHASE 2 — AUTOMATION INSTALL (THE 3-ENGINE SYSTEM)

### Engine 1 — SPEED TO LEAD (Government Bid Auto-Response)
**Problem it solves:** Contractor misses bid windows because they're in the field.
**What we install:**
- Automated SAM.gov / state portal bid alert system (daily digest)
- Pre-written bid response templates for NAICS 561720 / 562111
- Master Ledger bid tracking (open → submitted → awarded → closed)
- 48-hour turnaround protocol: alert → response → submission

**Pass criteria:** Client submits at least 2 government bids per month with zero manual search.

---

### Engine 2 — DOC PROCESSING (Audit-Readiness Automation)
**Problem it solves:** Client cannot pass a contract audit — no receipts, no SOPs, no documentation trail.
**What we install:**
- Master Ledger logic (revenue vs. expense categorization by NAICS)
- Digital receipt vault (Google Drive / OneDrive folder structure)
- Project Audit Log template (from `/Prime` skill — Prime Pathwy Compliance Protocol)
- Before/after photo documentation checklist for every job

**Pass criteria:** Client can produce a clean audit package for any job within 24 hours of request.

---

### Engine 3 — STATUS REPORTING (Ground Truth for the Owner)
**Problem it solves:** Owner has no real-time view of their business — flying blind.
**What we install:**
- Weekly Ground Truth Report (revenue, expenses, open jobs, bids in flight)
- Single dashboard view: cash position, DSCR analog, next 30-day forecast
- Escalation rule: any line item >$500 variance triggers an alert

**Pass criteria:** Owner can answer "What is my business worth right now?" in under 60 seconds.

## PHASE 3 — THE MIDDLEMAN BRIDGE

**The leverage move:** Use SB / DVBE / MBE certification status to win government contracts, then fulfill through certified subcontractors. The owner becomes the prime contractor — not the crew.

### How It Works
```
Prime Pathwy (Prime Contractor — SB Certified)
    ↓ wins contract via SB/DVBE preference
Certified Subcontractor (performs the work)
    ↓ billed at subcontractor rate
Prime Pathwy collects margin (15–30%)
    ↓ owner never touches a mop or hauls a load
```

### Certification Stack Required
| Cert | Status | Ledger Task |
|------|--------|-------------|
| CA Small Business (SB) | DATA LOCKED — submit to Cal eProcure | Task #03 |
| Federal MBE / SAM.gov | PENDING | Task #04 |
| NAICS 561720 in SAM | PENDING | Task #05 |
| NAICS 562111 in SAM | PENDING | Task #06 |

### Subcontractor Qualification Criteria
- Licensed + bonded in California
- Active Workers' Comp insurance on file
- Completed at least 3 documented jobs (photos + invoices)
- Signed Prime Pathwy Subcontractor Agreement (see `/legal/` template)
