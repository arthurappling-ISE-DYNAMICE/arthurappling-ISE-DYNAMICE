Understanding the Terminal Command
cd "C:\Users\arthu\GeminiEcosystem\PrimePathwyOS"

The command cd means change directory. You are telling the terminal:

"Take me into this folder so I can operate from inside it."

Without this command, the terminal does not know where your files are located.

06
The Lead Extraction Command
Import-Csv ".\research\leads\enriched_solano_crm_database.csv" |
Where-Object { $_.company_name -notlike "*Research*" } |
Format-Table company_name, sector, city, phone, website
Import-Csv
Opens and reads the spreadsheet
Where-Object
Filters rows using conditions
$_.company_name
Looks at the company_name column
-notlike "*Research*"
Removes placeholder research rows
Format-Table
Displays clean, readable output
