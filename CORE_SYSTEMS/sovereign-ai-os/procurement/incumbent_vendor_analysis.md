## DIRECT AWARD ELIGIBILITY INDICATORS

California allows direct awards (no competitive bid) under certain thresholds:

| Threshold Type | Amount | Notes |
|---|---|---|
| State Informal Bid (SB/DVBE) | Up to $250,000 | SB/DVBE set-aside eligible |
| County Informal Bid (general) | Varies by county ($5K–$100K) | Check each county's purchasing policy |
| Emergency/Disaster Cleanup | No threshold | Direct award common for disaster response |
| Piggyback Contracts | No threshold | Leverage existing state/county contracts |

**Strategic Recommendation:** Obtain SB and DVBE certifications through DGS OSDS to access set-aside contracts and informal bid opportunities below competitive threshold. This is the fastest path to initial government revenue.
