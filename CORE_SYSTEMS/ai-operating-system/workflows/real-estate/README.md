## Operational Standard

Every turnover engagement under this vertical must produce:
1. Before photos — minimum 8, all areas in scope
2. Completed 10-Point Inspection Checklist with PASS/FAIL per point
3. Disposal receipt (if haul-away in scope)
4. After photos matching before coverage
5. Written sign-off — crew lead + Prime Pathwy supervisor + client

No job is closed without all five artifacts on file.
