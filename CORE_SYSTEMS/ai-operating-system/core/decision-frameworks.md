## 1. OODA Loop — Operational Decision Cycle

Applied to all tactical decisions across contracting, betting intelligence, and capital deployment.

```
OBSERVE  → Gather raw data. No interpretation yet. Ground truth only.
ORIENT   → Apply context: DSCR gate, NAICS scope, set-aside eligibility, risk factors.
DECIDE   → Select one action. Document the decision and its basis.
ACT      → Execute with Exact Command + Pass Criteria + Error Map.
         → Loop: re-OBSERVE after action to confirm state change.
```

**Rule:** Do not skip from OBSERVE to ACT. ORIENT and DECIDE are mandatory.

## 2. DSCR Gate — Resource Deployment Filter

Before committing time, money, or system resources to any initiative:

```
Step 1: Estimate Output Value — specific, realistic, documented
Step 2: Estimate Input Cost   — time + direct spend + opportunity cost
Step 3: Calculate ratio       — Output Value / Input Cost
Step 4: Compare to threshold  — 7.42x required

≥ 7.42x → Proceed
< 7.42x → Restructure. Do not deploy until threshold is met.
```
