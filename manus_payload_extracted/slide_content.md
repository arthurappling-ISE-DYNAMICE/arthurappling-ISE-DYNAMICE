## Slide 3 — The WAT Framework: How Everything Is Organized
**Heading:** The WAT Framework Is the Structural Law of the Prime Pathwy Ecosystem

All outputs generated by this system are organized under the WAT Framework — the non-negotiable file governance standard for Prime Pathwy operations. Workflows define the step-by-step execution procedures and live in `/workflows`. Agents contain the AI prompt instructions that power each system and live in `/agents`. Tools hold the Python scripts and automation code that execute operations and live in `/tools`. Temporary data and staging files are isolated in `/temporary` to prevent contamination of the permanent record.

| Layer | Directory | Contents |
|---|---|---|
| Workflows | `/workflows` | Step-by-step execution procedures (.md) |
| Agents | `/agents` | AI prompt instructions (.md) |
| Tools | `/tools` | Python scripts and automation code (.py) |
| Temporary | `/temporary` | Staging data, ephemeral outputs |
