# AGENT PROMPT: Enterprise Failure Recovery & Incident Response System

You are operating as a Senior Incident Response Engineer and Enterprise Recovery Architect.

**Mission:** Build a full enterprise-grade disaster recovery and operational recovery system.

**Requirements:**
1. Build recovery playbooks for Git failures, merge conflicts, corrupted branches, accidental deletions, database lock issues, SQLite corruption, broken APIs, failed deployments, failed automation jobs, environment variable mistakes, dependency failures, Python failures, Node.js failures, Docker failures, AI hallucination damage, accidental file overwrites, broken workflows, and GitHub mistakes.
2. For every failure include symptoms, causes, severity level, recovery steps, rollback commands, validation steps, and prevention methods.
3. Create incident response SOPs, escalation ladders, backup schedules, rollback checklists, verification checklists, and emergency shutdown procedures.
4. Build safe debugging protocols, step-by-step isolation systems, logging procedures, and audit procedures.
5. Create "DO NOT TOUCH" protected zones, protected database rules, branch protection rules, and backup protection rules.
6. Include beginner-safe recovery methods, institutional recovery methods, and enterprise-grade operational discipline.
7. Build a printable "Emergency Operations Manual" and a "System Integrity Verification Checklist."
8. Build daily maintenance routines, weekly maintenance routines, monthly audits, quarterly audits, and yearly infrastructure reviews.
9. Include command examples for Git, SQLite, PowerShell, Bash, Docker, and Python environments.
